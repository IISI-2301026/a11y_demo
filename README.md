# 🌐 無障礙網頁開發實務指南

這是一個全面的無障礙網頁開發演示專案，展示了現代前端開發中的無障礙最佳實踐和新技術。

## 🚀 專案特色

### 基礎無障礙概念
- ✅ 語意化 HTML 結構
- ✅ 鍵盤導航支援
- ✅ 螢幕報讀器相容性
- ✅ 色彩對比度最佳化
- ✅ ARIA 標籤應用

### 現代無障礙技術
- 🗣️ **Web Speech API** - 語音朗讀功能
- 👁️ **Intersection Observer** - 視覺化閱讀進度
- 📱 **Media Queries** - 系統偏好設定偵測
- 🎯 **Focus Management** - 進階焦點陷阱
- ⚡ **Vue 3 Composition API** - 無障礙 Hooks

### 自動化測試工具
- 🧪 **Axe-core** 整合測試
- 🎨 **色彩對比檢測器**
- ⌨️ **鍵盤導航測試**
- 🔊 **螢幕報讀器模擬**
- 📊 **無障礙性能分析**

## 🛠️ 安裝與運行

### 快速開始
```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建置生產版本
npm run build
```

## 📋 無障礙檢查清單

### HTML 結構
- [ ] 使用語意化標籤
- [ ] 正確的標題層次結構
- [ ] 表單元素有對應的 label
- [ ] 圖片有適當的 alt 屬性

### ARIA 標籤
- [ ] 複雜元件使用適當的 role
- [ ] 動態內容使用 aria-live
- [ ] 表單驗證使用 aria-invalid

### 鍵盤操作
- [ ] 所有互動元素可透過 Tab 鍵訪問
- [ ] 焦點指示器清晰可見
- [ ] 支援 Enter/Space 啟動

### 視覺設計
- [ ] 文字與背景對比度 ≥ 4.5:1
- [ ] 不僅依賴顏色傳達資訊
- [ ] 支援系統偏好設定
